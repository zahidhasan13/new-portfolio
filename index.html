As a frontend developer, I specialize in crafting intuitive and visually stunning web applications using the latest technologies such as React, Redux, Tailwind CSS and Firebase. My primary focus is on creating responsive and user-friendly interfaces that provide an exceptional user experience across all devices. With a strong foundation in frontend development, I am committed to building applications that not only function flawlessly but also look great.

I have hands-on experience with modern web frameworks and libraries, and I thrive in environments that encourage clean, scalable, and efficient code. My skill set extends beyond just development, as I also bring a creative touch to my work with a keen eye for design. I am proficient in using tools like Figma for wireframing and prototyping, and I excel in enhancing user experiences with smooth animations using GSAP (GreenSock Animation Platform).

Combining my technical expertise with a passion for design, I create web applications that are both functional and aesthetically pleasing, delivering high-quality solutions that meet the needs of my clients.
